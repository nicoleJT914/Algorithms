带有平衡条件的BST

每个结点的左子树和右子树的高度最多差1
 
平衡的关键是在插入结点后，可能破坏原来的平衡，若将必须重新平衡的节点叫做x，则不平衡可能出现下面四种情况：
1. x ==> left ==> left ==> insert
2. x ==> right ==> right ==> insert
3. x ==> left ==> rifht ==> insert
4. x ==> right ==> left ==> insert

对于1和2情况，使用单旋转即可恢复平衡；对于3和4情况，需要使用双旋转才可恢复平衡

旋转后，要及时更新节点的高度值
